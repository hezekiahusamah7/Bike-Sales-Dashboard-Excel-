# Bike Sales Dashboard (Excel)

## Overview
This project demonstrates a complete data analysis workflow in Excel. Starting with raw customer data, I cleaned and organized the dataset, then used PivotTables to explore patterns in demographics, income, age, and commute distance. Finally, I built an interactive dashboard with charts and slicers to make insights easy to filter and understand.

## Key Insights
- Middle-aged customers purchased the most bikes.
- Higher income levels increased likelihood of purchase.
- Longer commute distances reduced bike sales.

## Tools
Microsoft Excel 2013 (Data Cleaning, PivotTables, Charts, Slicers)

## Outcome
A clean, interactive dashboard that highlights customer behavior and sales trends, showcasing skills in data preparation, analysis, and visualization.
